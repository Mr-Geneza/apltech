1. Развертывание проекта происходило на виртуальном сервере Open Server версии 5.4.3.
2. Версия PHP 8.0, MySQL 8.0, Apache 2.4 - PHP 8.0 - 8.1
3. Для получения всех товаров с backend-a, необходимо настроить сервер, а именно: прописать в htaccess следующий код
<IfModule mod_headers.c>
  Header set Access-Control-Allow-Origin "*"
</IfModule>
Без этого кода, сервер не позволит на frontend-e вытягивать данные с backend-a.
4. Для backend-а настроены пути показа, согласно тестовому заданию, с добавлением "backend" после localhost.
Пример: http://localhost/appltech/backend/api/products - все товары
5. Для работы frontend с backend, реализованы дополнительные методы, позволяющие вытаскивать данные в JSON формате.
6. По frontend выполнены следующие задания:
   1. Показ всех товаров
	2. Возможность просмотра выбранного товара
	3. Для марки Lenovo цена выводится из колонки rrp_price.
	4. Созданы методы для редактирования и удаления, но не до конца реализованы (физически не успел закончить).
7. Backend проект запускал через Open Server, frontend запускал командой npm run serve.
8. Если по каким-то причинам не получится запустить и проверить работу, свяжитесь со мной. В крайнем случае, могу снять видео, как всё работает, либо в онлайн режиме, всё посмотреть.
